# Obsidian Learning Coach Skill

一个轻量学习课件 skill。

它的边界很硬：

- AI 生成 `lessons/`：定义、原理骨架、边界、误区、画图任务、自检题。
- 用户生成 `notes/`：自己的解释、类比、图、复述、总结和疑问。
- AI 生成 `checks/` 或聊天反馈：只做事实核查，不润色、不重写。

这个 skill 不追求让 AI 把知识讲到顺滑可读。它追求让用户必须亲手重构理解。

## 适合的请求

```text
我想学 HTTPS，给我课件，不要给我笔记
```

```text
我完全不懂进程和线程，只给定义、原理和自检问题
```

```text
这是我自己写的笔记，请只检查事实错误，不要润色
```

## 默认目录

```text
LearningVault/
  topics/
    [topic]/
      lessons/
      notes/
      checks/
```

## lesson 是什么

`lessons/` 是 AI 产物，只放课件原料：

- 前置概念
- 核心定义
- 原理骨架
- 边界条件
- 易错点
- 结构图任务
- 自检问题

## note 是什么

`notes/` 是用户产物。AI 不代写。

用户在这里写：

- 自己的人话解释
- 自己的类比
- 自己画的图
- 自己的复述
- 自己的疑问
- 自己的错因

## fact check 是什么

当用户提交自己的笔记，AI 只检查：

- 定义是否准
- 因果关系是否反
- 图中箭头或顺序是否错
- 公式、单位、边界条件是否错
- 是否漏掉必要条件

如果无错，只回复：

```text
事实无误
```

如果有错，只列错误点，不提供重写版。

## 不做什么

默认不做：

- 长资料转换
- PDF 拆章
- 知识地图维护
- dashboard
- 自动复习计划
- 详细 concept notes
- AI 代写个人笔记

这些功能如果以后需要，可以作为独立 skill 或显式扩展重新加入。
