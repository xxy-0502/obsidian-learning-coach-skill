# Lightweight Progress Template

Progress is a state tracker, not a knowledge note.

````markdown
---
type: progress
topic: [主题]
created: YYYY-MM-DD
updated: YYYY-MM-DD
---

# [主题] 进度

- 用户 note：未写/已写/待核查/需修正
- 最近互动：
- 最近事实核查：
- 下一个动作：

## 待处理

- [ ] 写当前 lesson 对应的个人 note
- [ ] 完成下一次互动检查
- [ ] 修正 fact check 中的错误

## 未解决错误

## 轻量复看
````
