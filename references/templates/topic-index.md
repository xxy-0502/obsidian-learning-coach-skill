# Topic Index Template

`notes/[主题]/index.md` is a light course entry page, not a knowledge map or progress page. Keep only a few navigation links. Put concept relationships in `maps/[主题]知识地图.md`; put goals, mastery, review dates, mistakes, and next steps in `progress/[主题]/`.

```markdown
# [主题]

- 模式：[topic-first/source-first]
- 创建日期：YYYY-MM-DD

## 入口

- 知识结构：[[主题名知识地图|主题名 知识地图]]
- 当前课程：

## 说明

概念关系维护在 [[主题名知识地图|主题名 知识地图]]。
学习目标、掌握程度、复习安排和错题遗漏维护在 `progress/[主题]/`。
```

Replace `主题名` with the same safe topic/course name used for the topic folder.
